# 文件

按论文组织文件。每篇论文一个子文件夹，内部按类型分目录：

```
文件/
├── 基于机器学习的碳排放预测研究_Journal of Cleaner Production/
│   ├── 稿件/              ← 投稿论文 PDF/Word
│   ├── 附件/              ← 补充材料、数据等
│   ├── 审稿意见/           ← 编辑部/审稿人的审稿意见文件
│   └── 审稿意见回复/        ← 你对审稿意见的回复文件
└── 另一篇论文_期刊名/
    └── ...
```

文件通过稿子菌界面上传，自动按此结构组织。
